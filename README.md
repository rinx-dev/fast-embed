# Fast Embed

A pnpm workspace monorepo for framework-agnostic video embed primitives and framework wrappers.

## Packages

- `@fast-embed/core` contains provider adapters, thumbnail-first embed state, and iframe utilities.
- `@fast-embed/react` contains SSR-safe React hooks and components built on top of the core package.

## Commands

- `pnpm build`
- `pnpm test`
- `pnpm lint`
- `pnpm dev`
